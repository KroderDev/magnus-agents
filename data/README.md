# Pokémon knowledge

`cobblemon-knowledge.json` is a generated, compact runtime index for Cobbleverse
1.7.40. Spawn rules come from `COBBLEVERSE-DP-v30`; species profiles and
evolutions come from Cobblemon 1.7.3.

Regenerate it with `scripts/build-cobblemon-knowledge.ts` after extracting the
pack's `spawn_pool_world` and Cobblemon's `species` directories.
